# guide-cnsite-mk

本项目用于归档和发布多个独立 HTML 页面。仓库本身不针对任何特定域名或具体网站，仅作为静态页面集合的存放与展示空间。

## 项目简介

guide-cnsite-mk 是一个通用型 HTML 页面归档仓库。所有页面均为独立编写，内容涵盖各类主题，以方便查阅、分享和二次使用为目的。

仓库采用简洁的目录结构，每个页面独立存在，互不依赖，便于快速浏览和部署。

## 目录说明

```
guide-cnsite-mk/
├── index.html          # 首页/导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/             # 公共资源文件（如有）
    ├── css/
    └── js/
```

- `index.html`：仓库入口页面，提供页面列表或导航链接。
- `pages/`：所有独立 HTML 页面存放于此，文件名按内容或序号命名。
- `assets/`：存放页面可能共用的样式表、脚本等资源文件。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开。
- 页面内容不涉及任何具体商业、政治或敏感信息，仅供技术或知识参考。
- 页面可能引用外部资源（如字体、图标库），但尽量保持轻量和自包含。

## 维护说明

- 欢迎通过 Issue 提交页面建议或报告问题。
- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并在 `index.html` 中添加链接。
- 请确保新增页面不包含恶意代码、跟踪脚本或侵犯他人权益的内容。
- 本仓库由维护者不定期更新，不保证时效性或长期可用性。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在本地打开 `index.html` 浏览所有页面。
3. 也可将仓库部署到任意静态托管服务（如 GitHub Pages）在线访问。
